介绍

React

模块化：把可复用的一些代码，抽离成一个个模块，便于项目的开发和维护。

组件化：把一些可复用的UI元素，抽离为组件。

在移动app开发方面，react结合react-native，可以无缝迁移到移动app的开发。

几个核心概念

虚拟Dom（Virtual Document Object Model）

  • DOM的本质： 浏览器中，用js对象来表示页面上的元素，提供了操作DOM的API。

  • react中的虚拟DOM：在框架中，用JS对象来模拟页面上的DOM和DOM嵌套。

  • 为什么？ 为了实现页面中，高效更新。按需更新。

    • 获取新旧两个DOM树，然后对DOM树中的元素一一对比。元素改变时，更新。这样的操作减少了渲染的个数。

    假如要用js模拟以下的代码

    <div id="mydiv" title="mytitle">myContent</div>

    var div = {
  tagName: 'div',
  attrs: {
    id: 'mydiv',
    title: 'mytitle'
  },
  childrens: [
    'mycontent'
  ]
}
Diff算法
  • Tree diff: 新旧两个DOM树，逐层对比。
  • component diff: 在进行Tree Diff时，每一层中，组件的对比。如果对比，前后两个组件类型相同，则暂时认为此组件不需要被更新。 如果不同，移除旧的组件，创新新的组件并加到页面上。
  • element diff: 在组件对比的时候，如果组件类型相同，则需要对比其元素的对比。

Webpack

  1. 在项目跟目录中创建dist 和src 文件夹。src 目录下创建 index.html

  2. 安装webpack最好在本地安装而不是全局安装。全局安装可能会因为项目使用版本不同，会导致构建失败

    npm install --save-dev webpack

    对于webpack v4+ 版本需要另安装 CLI

    npm install --save-dev webpack-cli

  3. 运行webpack

    npx webpack

  4. 在webpack 4 以上版本，可以设置mode。 在项目的根目录，创建 webpack.config.js. Mode 一共有两种，开发(development)和产品(production);

    module.exports = {
    mode: 'development' //或者production
}

    development 的文件是开发格式化（未被压缩）。production是生成压缩后的文件

  5. 安装 webpack-dev-server

    可以用server来测试，不产生本地文件，储存在内存中，可高速读取。并减少一直压缩的次数，可以实时更新。

    npm install webpack-dev-server -D

  6. 配置 html-webpack-plugin 插件

    npm install html-webpack-plugin -D

    在 webpack.config.js文件中进行以下修改

    const path = require('path');
const HtmlWebPackPlugin = require('html-webpack-plugin');
   
const htmlPlugin = new HtmlWebPackPlugin({
    template: path.join(__dirname, './src/index.html'), //源文件
    filename: 'index.html' //生成在内存的首页
})
   
module.exports = {
    mode: 'development',
    plugins: [
        htmlPlugin
    ]
}

安装react并导入

需要安装两个包 react 和 react-dom

npm install react react-dom -S

其中 react 目的是创建组件，虚拟DOM元素，生命周期。

react-dom 是把创建好的组件和虚拟DOM放到页面上展示的

基本的渲染和嵌套

import React from 'react'
import ReactDOM from 'react-dom'


// 创建虚拟DOM元素
//参数1: 创建元素的类型，字符串，名称
//参数2: 对象或者null，当前DOM属性
//参数3： 字节点（其他DOM 获取 文本字节点）
//参数n： 其他子节点
// <h1 id="myh1" title="this is a h1">This is h1</h1>

const myh1 = React.createElement('h1',{id: 'myh1'},'this is h1')


//用react-dom 渲染
//参数1: 需要渲染的DOM元素
//参数2: 页面上的容器
ReactDOM.render(myh1,document.getElementById('app'));

//实现嵌套
const myh1 = React.createElement('h1',{id: 'myh1'},'this is h1')
const mydiv = React.createElement('div',null,'this is a div',myh1)
ReactDOM.render(mydiv,document.getElementById('app'));
在JS中，不能写类似于HTML的标记。但可以通过babel来转化. JS中混合写入HTML的语法，叫做JSX语法。

运行npm install babel-core babel-loader babel-plugin-transform-runtime -D

运行 npm install babel-preset-env babel-preset-stage-0 babel-preset-react -D

需要在根目录里创建 babelrc 来设置presets和插件

{
    "presets": ["env","stage-0","react"],
    "plugins": ["transform-runtime"]
}

需对webpack.config.js添加以下代码

module.exports = {
    mode: 'development',
    plugins: [
        htmlPlugin
    ],
    module: {//third party module setting rule
        rules: [
            { test: /\.js|jsx$/, use: 'babel-loader', exclude: /node_modules/}
        ]
    }
}
